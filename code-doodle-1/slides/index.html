Draw a line


							ctx.lineWidth = 10;
							ctx.strokeStyle = 'hsla(180, 100%, 50%, 1.0)';
							ctx.beginPath();

							// path starting point
							ctx.moveTo(0, 0);

							// path to here
							ctx.lineTo(200, 200);

							// draws the path with the current stroke style

Change a colour



							var count = 0;

							function step() {

								ctx.clearRect(0, 0, canvas.width, canvas.height);

								// ...
								ctx.strokeStyle = 'hsla(' + count + ', 100%, 50%, 1.0)';
								// ...

								count++;

								window.requestAnimationFrame(step);
							}

hsla description at MDN

https://mzl.la/2kbzOXk

The first argument to HSLA is the degrees, so our count will start at 0 which is red

when it loops around to 360 it turns red again, and so on

so, have a close look: red at 0 and then 360 again

Change position



							ctx.moveTo(x - 200, y - 100);

							var yCount = Math.sin((Math.PI / 180) * count)
							ctx.lineTo(x, y + yCount * 100);

							ctx.lineTo(x + 200, y);

if we treat count as angle in degrees, then

Math.sin((Math.PI / 180) * count)

will give us values between -1 and 1

because we won't see the animation if it only moves one pixel up or down

now we'll see our arm's elbow moving up and down
